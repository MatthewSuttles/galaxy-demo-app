This is a demonstration project for Galaxy SDK.

To compile this project you need the CMake utility.

Step by step configuration:
1. Download an appropriate development kit package from https://devportal.gog.com (you should register as a game developer at GOG first)
2. Unpack contents of the development kit package to the directory that contains CMakeLists.txt (or update DEVELOPMENT_KIT_DIRECTORY variable in CMakeLists.txt)
3. Configure and generate CMake build
4. You might want to change CLIENT_ID and CLIENT_SECRET in Game.cc to your own (you need to get it from GOG)
5. Compile
6. Run the Demo executable
